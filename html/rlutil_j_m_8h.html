File: rlutilJM About: Description Library for creating 2D Games for the Windows CMD. For exclusive use with Windows. Not compatible for versions lower than Windows vista.

Uses SFML for audio and networking properties. You should probably use the following guide to set up the proyect https://www.sfml-dev.org/tutorials/2.3/start-vc.php

This library utilizes rlutil from Tapio Vierros
